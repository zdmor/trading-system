## 一、周期体系：大-中-小三级联动

威科夫的核心逻辑之一：**大周期定方向，中周期找结构，小周期定入场**。

### 1.1 三级周期的标准配置

| 周期 | 级别 | 作用 | 常用时间框架 |
|------|------|------|------------|
| **大周期** | 战略 | 判断主要趋势方向（牛市/熊市/震荡） | 周线图 |
| **中周期** | 战术 | 识别吸筹/派发区间结构 | 日线图 |
| **小周期** | 执行 | 精确入场点（Spring/LPS） | 60分钟图 或 15分钟图 |

### 1.2 操作规则

```
规则1：大周期决定仓位方向
  周线上升趋势 → 只做多，不做空
  周线下降趋势 → 只做空，不做多
  周线横盘      → 多空皆可，轻仓试探

规则2：中周期决定操作框架
  日线出现吸筹结构 → 准备做多
  日线出现派发结构 → 准备做空
  日线无清晰结构 → 不操作

规则3：小周期决定精确入场
  60分钟/15分钟出现Spring/LPS/Upthrust → 执行入场
  60分钟/15分钟无确认信号 → 继续等待
```

## 二、选股体系

### 2.1 基本面初筛（剔除80%）

这是一个**排除系统**，不是打分系统：

| 排除项 | 标准 | 理由 |
|--------|------|------|
| ST/\*ST | 自动排除 | 财务风险 |
| 近1月解禁 > 总股本5% | 排除 | 抛压风险 |
| 质押率 > 50% | 排除 | 爆仓风险 |
| 近两季营收/利润均下滑 | 排除 | 基本面恶化 |
| ROE < 8%（连续两年） | 排除 | 盈利质量差 |
| 商誉/净资产 > 30% | 排除 | 减值风险 |
| 日均成交额 < 5000万 | 排除 | 流动性差 |

### 2.2 技术面精选——寻找结构

在通过基本面初筛的股票中，用日线图寻找以下**两种结构之一**：

#### 结构一：吸筹区（准备做多）

```
吸筹区识别标准（五项全部满足）：
1. 之前有一段明确的下跌趋势（≥20%跌幅）
2. 价格在一个区间内横盘 ≥ 3周
3. 区间内出现过一次放量暴跌后的快速反弹（SC+AR）
4. 横盘后期，价格缩量测试支撑位不破（ST缩量）
5. 区间内均线逐渐走平并开始向上收敛
```

#### 结构二：再吸筹区（上涨中继，准备加仓）

```
再吸筹区识别标准：
1. 股票已经处于明确的上升趋势中
2. 回调幅度 ≤ 前期涨幅的50%
3. 回调缩量，触及前支撑位或均线
4. 形成一个小型横盘区间（1-3周）
5. 均线多头排列未被破坏
```

### 2.3 选股优先级排序

```
优先级 = 结构完成度 × 区间时长 × 大盘共振度

- 吸筹区≥5周 + 大盘也在筑底 → 最高优先级
- 再吸筹区 + 大盘趋势向上 → 次高优先级
- 吸筹区≥3周但均线未收敛 → 中等优先级，继续观察
- 区间<3周 → 结构不充分，放弃
```

## 三、技术信号体系：定量量价信号清单

威科夫方法的灵魂是**量价关系**。以下每个信号都有明确的量价条件。

### 3.1 吸筹阶段信号（Phase A→E）

#### Phase A：下跌停止信号

| 信号 | 条件 | 含义 |
|------|------|------|
| **PS（初步支撑）** | 价格创阶段新低后突然放量反弹，成交量 ≥ 前20日均量的1.5倍 | 首次有大资金进场 |
| **SC（恐慌抛售）** | 当日跌幅 ≥ 4%，成交量 ≥ 前20日均量的2倍，K线有长下影线（下影线长度 ≥ 实体2倍） | 散户割肉，主力接盘 |
| **AR（自动反弹）** | SC后2-5天内反弹幅度 ≥ SC跌幅的50%，成交量逐日递减 | 卖盘暂时枯竭 |
| **ST（二次测试）** | 价格再次回落到SC低位附近，但成交量 ≤ SC当日成交量的50% | **最重要信号**：卖盘真的枯竭了 |

**操作**：Phase A阶段**不建仓**，只观察记录。画出AR高点和SC低点作为区间上下轨。

#### Phase B：建仓区

| 信号 | 条件 | 含义 |
|------|------|------|
| **缩量横盘** | 价格在AR和SC之间震荡，成交量持续低于20日均量 | 主力在低位慢慢吸筹 |
| **均线收敛** | 20日/60日均线间距持续缩小，最终走平 | 成本趋于集中 |

**操作**：Phase B中后期可以**轻仓试探（≤10%仓位）**，止损设在SC低点下方2%。

#### Phase C：弹簧测试（最关键信号）

| 信号 | 条件 | 买入动作 |
|------|------|---------|
| **Spring（弹簧）** | ①价格跌破SC/ST低点 ②当日或次日收回支撑位之上 ③跌破时缩量（≤ST当日量的70%）④收回时小幅放量 | **主力建仓（仓位30%）** |
| **强力Spring** | ①跌破后当日长下影线收回（下影线≥实体2倍）②跌破时短暂放量后快速缩量 ③收回时量能不大但K线饱满 | **重仓进场（仓位50%）** |

**Spring 信号的量价核心**：
> "Spring的跌破必须是假的——跌破时没有真实的卖盘，收回时轻盈而果断。如果跌破时放巨量且收不回来，那是真破位，不是Spring。"

**Spring的量化确认**：
```
假破位确认标准（3项全部满足=Spring确认）：
1. 破位当日或次日收回（收盘价回到支撑位上方）
2. 破位过程成交量 ≤ SC当日成交量的50%
3. 收回时K线为阳线，收盘价 ≥ 当日最高价的70%位置
```

#### Phase D：确认与突破

| 信号 | 条件 | 买入动作 |
|------|------|---------|
| **SOS（强势信号）** | 价格放量突破AR高点（区间上轨），成交量 ≥ 前20日均量的1.5倍 | **加仓至目标仓位** |
| **LPS（最后支撑点）** | SOS突破后，价格回踩原区间上轨（阻力变支撑），回踩时缩量 | **最后一次加仓机会** |

**量价确认**：
- SOS突破时必须是**真突破**：放量+收盘价稳稳在上轨上方
- LPS回踩必须是**假回踩**：缩量（≤突破日量的50%）+不跌破原上轨

#### Phase E：趋势运行

| 信号 | 操作 |
|------|------|
| 沿20日均线上涨，缩量回调到均线反弹 | 持仓不动，追踪止损上移 |
| 出现放量长阴跌破20日均线 | 减仓50% |
| 出现高位放量滞涨（3日创新高但量缩） | 减仓80% |

### 3.2 派发阶段信号（做空的镜像）

**派发是吸筹的镜像**，信号对称但方向相反：

| 吸筹信号 | 对应派发信号 | 条件 |
|---------|------------|------|
| SC（恐慌抛售） | **BC（买入高潮）** | 价格急涨 ≥ 5%，成交量 ≥ 前20日均量2倍，上影线长 |
| AR（自动反弹） | **AR（自动下跌）** | BC后2-5天回落 ≥ BC涨幅50% |
| ST（二次测试） | **ST（二次测试）** | 回到BC高位附近，成交量 ≤ BC量的50% |
| Spring | **Upthrust（上冲）** | 短暂突破区间上轨后快速回落，突破时缩量 |
| Spring 假跌破 | **UTAD（上冲后派发）** | Upthrust后出现放量长阴 |

**做空信号**：
- **Upthrust确认** = 空头建仓（同Spring做多的镜像条件）
- **SOW（弱势信号）= 放量跌破区间下轨** = 加仓做空
- **LPSY（最后供给点）= 反弹到原下轨（支撑变阻力）缩量** = 最后加仓做空机会

---

### 3.3 量价关系速查表（核心工具）

这是威科夫最核心的判断工具——每根K线都要看量：

| K线形态 | 成交量 | 判断 | 操作 |
|---------|--------|------|------|
| 大阳线 | 放量 | **真需求**，多头力量强 | 看多 |
| 大阳线 | 缩量 | **需求不足**，上涨乏力 | 警惕 |
| 大阴线 | 放量 | **真供给**，空头力量强 | 看空 |
| 大阴线 | 缩量 | **供给不足**，下跌无力 | 警惕 |
| 小K线（十字星/小阴小阳）| 缩量 | **无方向**，平衡状态 | 观望 |
| 小K线 | 放量 | **努力无结果**，主力在转移筹码 | **高度警惕反转** |
| 长下影阳线 | 缩量 | **需求吸收供给**，底部长下影=Spring | 买入信号 |
| 长上影阴线 | 缩量 | **供给压倒需求**，顶部长上影=Upthrust | 卖出信号 |

**最重要的两个量价背离信号**：
1. **高位放量滞涨**（新高但量缩 + K线小）= 派发 → 卖出
2. **低位缩量止跌**（新低但量缩 + 快速收回）= 吸筹/Spring → 买入

## 四、出入场体系

### 4.1 做多入场点（按优先级排序）

| 优先级 | 入场点 | 确认条件 | 初始仓位 |
|--------|--------|---------|---------|
| ★★★★★ | Phase C Spring后收回 | 3项确认全部满足（见上） | 30%-50% |
| ★★★★ | Phase D LPS回踩确认 | 缩量回踩原上轨不破 | 加至目标仓位 |
| ★★★ | Phase D SOS突破 | 放量突破AR高点确认 | 加至目标仓位 |
| ★★ | Phase B 缩量横盘后半段 | 均线收敛+量缩至极低 | ≤10%（试探） |

### 4.2 做空入场点（镜像）

| 优先级 | 入场点 | 确认条件 | 初始仓位 |
|--------|--------|---------|---------|
| ★★★★★ | Phase C Upthrust后回落 | 假突破后放量回落 | 30%-50% |
| ★★★★ | Phase D LPSY反弹确认 | 缩量反弹到原下轨不破 | 加至目标仓位 |
| ★★★ | Phase D SOW跌破 | 放量跌破区间下轨 | 加至目标仓位 |

### 4.3 止损设置

```
止损位置（由紧到松三个级别）：
1. 保守止损：Spring最低点下方 2%
2. 标准止损：SC最低点下方 2%
3. 宽松止损：整个吸筹区间下轨下方 3%

止损移动规则：
- SOS突破后 → 止损上移至AR高点下方 2%
- LPS确认后 → 止损上移至LPS低点下方 2%
- 趋势运行中 → 止损上移至20日均线下方 2%（移动止损）
```

### 4.4 止盈设置

**目标测算（因果法则）**：
```
方法一：区间宽度法
  目标价 = 区间上轨 + (区间上轨 - 区间下轨)
  即：区间宽度 × 1 = 最小目标

方法二：成交量加权法
  目标价 = 区间上轨 + (成交量最大日的振幅 × 2)

方法三：分批止盈法（推荐）
  第一目标（区间宽度的50%）：减仓30%
  第二目标（区间宽度的100%）：减仓40%
  第三目标（让剩余30%跑趋势，追踪止损保护）
```

### 4.5 出场信号

| 出场信号 | 条件 | 动作 |
|---------|------|------|
| 目标止盈 | 达到目标价 | 按比例分批出场 |
| 高位量价背离 | 连续3日创新高但量缩至前20日均量的60%以下 | 减仓50% |
| 高位放量长阴 | 单日跌幅 ≥ 4%，成交量 ≥ 前20日均量2倍 | 清仓 |
| 跌破追踪止损 | 收盘价跌破20日均线 | 清仓 |
| 出现派发结构 | 日线出现BC+UT等派发信号 | 清仓，准备做空 |

## 五、仓位管理体系

### 5.1 金字塔建仓法

```
完整做多仓位建设流程（以100万账户为例）：

Step 0：账户总风险控制
  单笔交易最大亏损 = 总资产 × 2% = 2万元
  单笔交易最大仓位 = 总资产 × 50% = 50万元

Step 1：试探建仓（Phase B末期）
  仓位：10%（10万元）
  买入价：区间中位
  止损价：SC低点下方2%
  最大亏损：(买入价-止损价)/买入价 × 10万

Step 2：Spring确认建仓（Phase C）
  仓位：追加至30%（30万元）
  买入价：Spring收回当日收盘价
  止损价：Spring最低点下方2%

Step 3：SOS突破加仓（Phase D）
  仓位：追加至50%（50万元，达最大仓位）
  买入价：SOS突破日收盘价
  止损价：上移至AR高点下方2%

Step 4：移动止损（Phase E）
  止损逐步上移至20日均线
  永不增加仓位（趋势中不加仓）

总结：
  10% → 30% → 50%（金字塔加仓，初期仓位轻，确认后加重）
  ⚠️ 绝对禁止：50% → 30% → 10%（倒金字塔，越涨越买=自杀）
```

### 5.2 多品种仓位分配

```
账户持仓分配规则：
- 同时持仓不超过3个品种
- 单个品种最大仓位 ≤ 50%
- 同板块品种总仓位 ≤ 60%
- 现金始终保留 ≥ 30%（用以应对回撤时的补仓机会）
```

### 5.3 追踪止损规则

```
追踪止损上移节奏：
Phase C → 止损在Spring低点下方
Phase D SOS后 → 止损上移至AR高点下方
Phase D LPS后 → 止损上移至LPS低点下方
Phase E 趋势中 → 止损每日上移至20日均线（只上移不下移）

例外规则：
  如果浮盈已超过目标价的80%，可以手动锁定全部利润，
  不等待追踪止损触发。
```

## 六、情绪面监控体系

威科夫的核心洞察之一：**比价格更重要的是价格背后的人在做什么**。

### 6.1 主力资金监控（A股实操）

| 指标 | 数据来源 | 信号 | 含义 |
|------|---------|------|------|
| 主力净流入连续方向 | 金融数据/同花顺 | 连续5日净流入 | 主力建仓 |
| 融资余额变化 | 两融数据 | 温和增长（非急剧放大）| 散户跟风适度 |
| | | 急剧放大（1周增30%+）| 散户过热，警惕 |
| 股东户数 | 公司公告（季报） | 环比减少 > 5% | 筹码集中（吸筹） |
| | | 环比增加 > 10% | 筹码分散（派发） |
| 大宗交易 | 交易所公告 | 折价率 > 5% + 大额 | 主力出货 |
| | | 溢价成交 | 主力看好 |
| 机构调研 | 公告/新闻 | 近期多家机构密集调研 | 关注度上升 |

### 6.2 市场情绪指标

| 指标 | 信号（危险） | 信号（机会） |
|------|------------|------------|
| 全市场涨停家数 | > 200家 | < 30家 |
| 全市场跌停家数 | > 100家 | < 5家且非恐慌 |
| 融资买入额/总成交 | > 12% | < 6% |
| 银证转账净流入 | 大幅持续增加 | 大幅持续减少 |
| 新开户数 | 月增 > 300万 | 月增 < 100万 |
| 基金发行规模 | 单月 > 2000亿 | 单月 < 200亿 |

### 6.3 情绪面的威科夫解读

```
当情绪面对应吸筹区：
  - 散户悲观、媒体唱空、新开户极少 → 正常
  - 散户已经开始乐观讨论 → 吸筹已进入末期，即将突破

当情绪面对应派发区：
  - 散户狂热、媒体唱多、新开户暴增 → 典型派发特征
  - 融资余额急剧放大、基金爆款频出 → 顶部信号

核心规则：
  "在别人贪婪时恐惧，在别人恐惧时贪婪"
  威科夫体系中 = "在散户派发时做空，在散户吸筹（割肉）时做多"
```

## 七、综合决策矩阵：五维评分系统

每笔交易前，用这个矩阵打分。**低于60分不操作**。

| 维度 | 考察项 | 满分 | 评分标准 |
|------|--------|------|---------|
| **基本面**（20分） | 通过初筛（无排除项） | 10分 | 全部通过=10，1项不通过=0 |
| | 行业处于景气周期 | 5分 | 明确上行=5，中性=2，下行=0 |
| | ROE > 15%（连续2年） | 5分 | 是=5，否=0 |
| **技术面**（25分） | 日线出现清晰的结构 | 15分 | 吸筹/派发结构完整=15，部分=8，无=0 |
| | Spring/Upthrust信号确认 | 10分 | 3项确认全满足=10，部分=5，无=0 |
| **量价面**（20分） | 量价关系健康 | 10分 | 放量上涨缩量回调=10，偶尔异常=5，量价背离=0 |
| | 缩量测试（ST）缩量明显 | 10分 | 缩量至SC的50%以下=10，50-80%=5，>80%=0 |
| **情绪面**（15分） | 主力资金流向支持 | 8分 | 连续5日净流入=8，平衡=4，连续流出=0 |
| | 散户情绪反向 | 7分 | 散户恐慌/悲观=7，中性=4，散户亢奋=0 |
| **仓位面**（20分） | 盈亏比 ≥ 3:1 | 10分 | ≥3=10，2-3=5，<2=0 |
| | 单笔风险 ≤ 总资产2% | 10分 | ≤2%=10，2-3%=5，>3%=0 |

**操作规则**：
- ≥ 80分：标准仓位操作（金字塔50%）
